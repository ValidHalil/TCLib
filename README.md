# EN: TCLib for Draw.io

A library of network equipment designed to quickly create drawings of telecommunication cabinets. 
## Features

- Extremely detailed device images created from scratch.
- All devices are grouped elements, which, if necessary, can be disassembled into components.
- Quick cabinet placement and drawing design due to prepared footnotes and frames.
- Modular storage system for device types.

## Installation

1. Download the modules folder and import all or only the necessary ones.

2. Alternatively, download the full library as a single file (not recommended due to the size of the images!).

# RU: TCLib для Draw.io

Библиотека сетевого оборудования, предназначенная для быстрого создания чертежей телекоммуникационных шкафов. 
## Особенности

- Крайне детально проработанные изображения устройств, созданные с нуля.
- Все устройства представлят собой сгруппированные элементы, которые, в случае необходимости возможно разобрать на составляющие.
- Быстрое расположение в шкафу и оформление чертежа за счет заготовленных сносок и рамок.
- Модульная система хранения типов устройств.

## Установка

1. Загрузить папку с модулями и импортировать все или только необходимые.

2. В качестве альтернативы загрузить полную библиотеку одним файлом (не рекомендуется из-за размера изображений!).
